# FlatttenFolder
a shell script to flatten a folder

flattenfolder.bat
This batch move all files and subfolders under specified folder to the root folder itself (virtually eliminate the direct children folders). 

flatten folder.reg
This registry file create a shell right click menu named Flatten Folder when imported to Windows. Then the user will be able to issue command via a quick click from right-click menu to flatten a folder. Note the content of this file must be updated to reflect the actual install location of the previous batch file.
